* Quick Start
** Install Clojure
*** Requirements
    - bash
    - curl
    - rlwrap
    - java
*** Arch Linux
#+BEGIN_SRC bash
sudo pacman -S clojure
#+END_SRC    
*** Linux    
#+BEGIN_SRC bash
curl -O https://download.clojure.org/install/linux-install-1.9.0.358.sh
chmod +x linux-install-1.9.0.358.sh
sudo ./linux-install-1.9.0.358.sh
#+END_SRC
*** OSX
#+BEGIN_SRC bash
brew install clojure
#+END_SRC
** Install NPM Dependencies
#+BEGIN_SRC bash
yarn install
#+END_SRC
